## Overview

This project is a WordPress-based directory platform built to organize and help users discover Telegram groups, channels, and bots. The main objective was to deliver a fast, accessible, and SEO-oriented experience through a well-structured information architecture and an efficient discovery system.

I handled the project end-to-end, from initial concept and build to performance optimization and technical refinement.

## Key Features

- **Structured Directory Architecture**: Clear categorization and listing structure to support discoverability and scalable content growth.
- **Search & Filters**: Dedicated search and filtering experience to quickly surface relevant groups, channels, and bots.
- **Submission Workflow**: Frontend submission forms enabling content contributions while maintaining data consistency.
- **Manual Moderation**: Controlled publishing flow with review and moderation to ensure quality and reduce spam.
- **Performance-First Build**: Implemented multiple optimization layers to keep the platform fast at scale.

## Technologies Used

- **WordPress Development**
- **Performance Optimization**
- **SEO Performance**
- **Directory Submission**
- **Filters**
- **LiteSpeed Cache**
- **WebP Images**
- **Lazy Loading**
- **CSS/JS Optimization**

## Challenges and Learnings

A key challenge was keeping performance high while supporting dynamic directory functionality such as search, filters, and submission flows. This required a careful balance between feature richness and front-end efficiency, plus continuous tuning of caching, asset delivery, and media handling to preserve strong PageSpeed results.

## Outcome

The final result is a fast, scalable, and SEO-oriented directory website optimized for discoverability and usability. The platform achieved 90+ PageSpeed scores on mobile and 100 on desktop through a combination of caching, image optimization, lazy loading, and CSS/JS refinement.
