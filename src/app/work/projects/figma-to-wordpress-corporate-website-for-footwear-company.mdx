## Overview

This project involved the development of a corporate website for a footwear company as part of an agency collaboration. The goal was to translate a provided Figma design into a clean, structured, and professional WordPress website tailored for B2B communication.

I focused on frontend implementation, ensuring consistency with the original design while maintaining clarity, usability, and scalability.

## Key Features

- **Figma to WordPress Implementation**: Accurate translation of the provided Figma layouts into a structured WordPress frontend.
- **Elementor-Based Frontend Development**: Use of Elementor to build flexible and maintainable layouts aligned with the design system.
- **Responsive Refinement**: Careful optimization across desktop, tablet, and mobile devices.
- **Multilingual Setup**: Italian and English language implementation with language switching integrated into the main navigation.
- **Hosting & Deployment**: Hosting configuration and final website deployment.

## Technologies Used

- **WordPress Development**
- **Figma**
- **Front-End Development**
- **Responsive Design**
- **Multilingual Translation**

## Challenges and Learnings

One of the main challenges was preserving design consistency while adapting a corporate-oriented layout to different screen sizes and languages. Managing multilingual navigation and content structure reinforced the importance of clarity and hierarchy in B2B website development.
