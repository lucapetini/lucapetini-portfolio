## Overview

This project involved the redesign and full rebuild of a corporate website for a construction company, developed from scratch in close collaboration with the client. The objective was to deliver a clean and professional institutional presence tailored to B2B clients and public administrations.

I handled the entire process, from UX structure and visual design to WordPress development using Elementor Pro, ensuring long-term scalability and ease of management.

## Key Features

- **End-to-End Website Rebuild**: Complete redesign and development from UX planning to final deployment.
- **Elementor Pro Development**: Modular layout built with Elementor Pro to support future content expansion.
- **Institutional UX Design**: Clear information architecture focused on usability and professional communication.
- **Performance-Oriented Structure**: Lightweight, optimized build designed for speed and reliability.
- **SEO-Ready Setup**: Clean URLs, structured content, and best practices to support organic visibility.
- **Security & Manageability**: Basic security hardening and a backend optimized for client autonomy.

## Technologies Used

- **WordPress Development**
- **Elementor**
- **Responsive Design**
- **Website Security**
- **WordPress Website Design**

## Challenges and Learnings

A key challenge was aligning institutional design requirements with performance and scalability goals. Building a modular structure that remained simple for internal teams to manage reinforced the importance of UX clarity, component reuse, and clean technical foundations in corporate WordPress projects.

## Outcome

The final result is a professional, scalable, and easy-to-manage corporate construction website. The project highlights strong capabilities in full WordPress builds, UX-driven design, and performance-oriented development for institutional and B2B contexts.
